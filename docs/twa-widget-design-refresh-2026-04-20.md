# TWA Quick Log Widget Design Refresh

## 목적
- Android TWA 홈 화면 위젯의 시각 밀도를 낮추고, ottline 브랜드 톤을 더 분명하게 반영한다.
- 기존 `기록하기`, `타임라인` 빠른 진입 기능은 유지한다.

## 범위
- 대상: `apps/twa` Android widget 리소스
- 영향: UI만 변경
- 비영향: API, DB, Sync, 웹 앱 라우팅, 위젯 액션 로직

## 변경 방향
1. 기존 1행 가로 2버튼 구조를 2x2 카드형 위젯으로 바꾼다.
2. 상단에 `OTTLINE` 배지와 설명 문구를 배치해 위젯만 봐도 용도를 이해할 수 있게 한다.
3. 액션 버튼은 세로 스택 카드로 바꿔 터치 영역과 가독성을 높인다.
4. record/timeline 액션에 서로 다른 배경 톤을 써서 시선 분리를 만든다.

## 검증 시나리오
1. 홈 화면에 위젯 추가 시 2x2 크기로 정상 배치되고 텍스트가 잘리지 않는다.
2. `기록하기` 탭 시 기존과 동일하게 `/?quick=1&quick_focus=1` 경로로 열린다.
3. `타임라인` 탭 시 기존과 동일하게 `/timeline`으로 열린다.
4. 위젯 리사이즈 시 배경/간격이 깨지지 않고 액션 카드가 유지된다.
