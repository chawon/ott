- 패키지명: `kr.pe.preview.ott.twa`
- 앱명: `On the Timeline (OTT)`
- 기본 언어: 한국어 (ko-KR)
- 대상 도메인: `https://ott.preview.pe.kr`
- 개인정보처리방침 URL: `https://ott.preview.pe.kr/privacy`

## 텍스트 초안
- 앱 이름: `온더타임라인 OTT`
- 짧은 설명: `영상과 독서 기록을 빠르게 남기고, 타임라인으로 되돌아보세요.`
- 상세 설명:
  `온더타임라인(OTT)은 영상과 책 기록을 한곳에서 관리하는 개인 기록 서비스입니다.

  작품을 검색해 기록을 남기고, 시간순 타임라인에서 내가 본 것과 읽은 것을 한눈에 정리할 수 있습니다.

  주요 기능
  - 영상/도서 작품 검색 후 기록 작성
  - 내 기록 타임라인 확인
  - 웹과 PWA 기반으로 빠른 접근
  - 간결한 입력 흐름으로 기록 부담 최소화

  이런 분께 추천해요
  - 콘텐츠 감상 기록을 꾸준히 남기고 싶은 사용자
  - 영상과 독서 이력을 함께 관리하고 싶은 사용자
  - 복잡한 기능보다 빠른 기록 경험이 중요한 사용자`

## 에셋 체크리스트
- [ ] 앱 아이콘 (512x512 PNG, 32-bit, alpha 가능)
- [ ] 피처 그래픽 (1024x500 PNG/JPG, 텍스트 과다 금지)
- [ ] 휴대폰 스크린샷 2~8장 (권장 1080x1920 또는 1242x2208)
- [ ] 다크/라이트 주요 화면 각각 1장 이상 포함
- [ ] 기록 작성 흐름 + 타임라인 확인 흐름 포함

## 캡처 권장 시나리오
1. 홈 화면 (서비스 가치가 보이는 화면)
2. 작품 검색 화면 (영상/도서 선택 가능 상태)
3. 기록 작성 화면 (입력 UI 명확히 보이게)
4. 타임라인 화면 (저장 결과 확인)
5. 설정 화면 (개인화/모드 옵션)
